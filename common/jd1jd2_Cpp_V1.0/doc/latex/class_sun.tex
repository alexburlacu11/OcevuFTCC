More accurate solar ephemeris. 

Implemenataion of Jean Meeus' more accurate solar ephemeris. \par
 For ultimate use in helio correction! From Astronomical Formulae for Calculators, pp. 79 ff. This gives sun's position wrt {\itshape mean} equinox of date, not {\itshape apparent}. Accuracy is $<$$<$ 1 arcmin. Positions given are geocentric. Parallax due to observer's position on earth is ignored. This is up to 8 arcsec; Routine is usually a little better than that. \par
 // -- topocentric correction {\itshape is} included now. -- // \par
 Light travel time is apparently taken into account for the ra and dec, but the author don't know if aberration is and he don't know if distance is simlarly antedated.
